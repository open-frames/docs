# XMTP

Open Frames is supported by XMTP clients such as [Converse](https://getconverse.app/) and [XMTP Chat](https://xmtp.chat/).

## Reference

See the [XMTP Frames docs site](https://xmtp.org/docs/build/frames#protocol-compatibility) for more information on how to use Open Frames with XMTP.
